# kube-memory client

Web dashboard and landing page for kube-memory.

See the [root README](../README.md) for product overview and setup. API reference: [server/API_DOC.md](../server/API_DOC.md).

```bash
npm install
npm run dev
```

Runs at **http://localhost:5173** with the API proxied to `http://localhost:3000` in development.
